# Bestellseite

Eine einfache E-Commerce-Bestellseite, ähnlich wie Amazon.

## Funktionen
- Produktliste anzeigen
- Produkte zum Warenkorb hinzufügen
- Warenkorb anzeigen und verwalten
- Einfacher Checkout-Prozess

## Wie man es ausführt
1. Stelle sicher, dass Python installiert ist.
2. Führe `python3 -m http.server 8000` im Projektverzeichnis aus.
3. Öffne http://localhost:8000 in deinem Browser.

## Dateien
- `index.html`: Haupt-HTML-Datei
- `style.css`: CSS-Styling
- `script.js`: JavaScript für Funktionalität
